# Briefly.ai
This is the ai based browser extension for summary of live web page
## To make it work follow these step 
>>Add the dist folder in the browser extension by load extension locally. 
>>dowload Ollama  from the its official website 'https://ollama.com/'.
>>After downloading the ollama run it.
>>Open CMD run the command "Ollama --version" to verify the installation.\
##Install any ai model for the summary perpose according to you GPU capability.
>>Add the name of the ai model to the url in .env  file.
## You are good to go.
